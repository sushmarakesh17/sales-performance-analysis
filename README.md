# Sales Performance Analysis

## Project Overview

This project analyzes sales data to identify business performance using Python, Pandas, Matplotlib, and Streamlit.

## Features

- Total Revenue
- Total Profit
- Total Orders
- Average Order Value
- Monthly Revenue Trend
- Top Selling Products
- Sales by Region
- Profit by Category
- Complete Sales Dataset

## Technologies Used

- Python
- Pandas
- Matplotlib
- Streamlit

## Dataset

sales_data.csv

Columns:
- Order ID
- Date
- Product
- Category
- Region
- Quantity
- Price
- Profit

## Run the Project

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
